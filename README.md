# Saukhyam – Personal Health & Wellness Platform

Saukhyam is a web-based personal health and wellness platform designed to support women’s physical and mental well-being. It aims to provide accessible health guidance, awareness, and basic wellness support through a simple and user-friendly interface. This project was developed as part of a hackathon initiative with a focus on holistic care, accessibility, and practical implementation.

## Features

- AI-powered conversational health assistant
- Appointment booking and viewing system
- Dedicated health sections:
  - Menstrual health
  - Pregnancy care
  - Post-pregnancy care
  - Mental health
  - General wellness tracking
- Health awareness library
- Responsive UI using HTML, CSS, and JavaScript

## Tech Stack

- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Data Storage: JSON-based local storage
- Deployment Platform: Render
- Version Control: Git and GitHub
